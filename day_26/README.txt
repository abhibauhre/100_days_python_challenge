# Personal Productivity Tracker - Usage Instructions

This project demonstrates all major Python functionalities you've learned:
- File I/O (JSON, CSV)
- List/dict comprehensions
- Error handling
- Functions and OOP
- API usage (motivational quotes)
- pandas for data analysis (optional)

How to use:
1. Run `main.py` in this folder.
2. Use the menu to add, list, and complete tasks.
3. Save your tasks to CSV for spreadsheet use.
4. Analyze your productivity by day (if pandas is installed).
5. Get a motivational quote from the internet.

You can install pandas with:
    pip install pandas

All data is saved in `tasks.json` and `tasks.csv` in this folder.

---

Feel free to extend this project with more features, such as deadlines, priorities, or visualizations!
